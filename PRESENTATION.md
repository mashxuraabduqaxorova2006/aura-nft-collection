# AuraNFT Project Presentation 🛡️✨

## 1. Project Overview
**AuraNFT** is a state-of-the-art DApp that allows users to mint unique digital entities and engage in a decentralized voting system.
- **Problem**: Most NFT platforms are too complex for casual collectors.
- **Solution**: A premium, easy-to-use interface with built-in community engagement (Voting).

## 2. Technical Stack
- **Smart Contract**: Solidity (ERC721URIStorage), OpenZeppelin.
- **Frontend**: HTML5, Premium Vanilla CSS (Glassmorphism), JavaScript.
- **Blockchain Interface**: Ethers.js v5.7.
- **Image Assets**: AI-generated premium cosmic visuals.

## 3. Key Features
- **Minting**: Direct on-chain NFT creation with metadata.
- **Voting System**: Decentralized "Like" system for digital assets.
- **Responsive UI**: Optimized for all devices with high-end animations.
- **Secure**: Built using industry-standard libraries.

## 4. Contract Architecture
- Inherits from `ERC721URIStorage` for flexible metadata storage.
- Features custom `voteCount` mapping and `hasVoted` security check.
- Emits real-time events for frontend synchronization.

## 5. Future Roadmap
- Integration with IPFS/Arweave for decentralized storage.
- Marketplace functionality (Buy/Sell/List).
- DAO integration for collective collection management.
